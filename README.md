## 📖 ¿Qué es JOMAR?

**JOMAR S.A.S.** es una empresa de **logística y transporte** especializada en La Guajira, Colombia. Ofrecemos soluciones integrales de transporte, almacenamiento y distribución de mercancías con más de 10 años de experiencia sirviendo a más de 15 municipios.

Nuestra misión es brindar servicios logísticos confiables, eficientes y de calidad, combinando la excelencia empresarial con el respeto a la **identidad cultural Wayuu**.

Esta landing page es el escaparate digital oficial de JOMAR, diseñada para presentar nuestros servicios, experiencia y valores a clientes potenciales.

## 🚀 Cómo Usar

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/LaszloSM/jomar.git
cd jomar

# Instalar dependencias
npm install
```

### Desarrollo

```bash
# Iniciar servidor de desarrollo
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) para ver la landing page en desarrollo.

### Production

```bash
# Build para producción
npm run build

# Iniciar servidor de producción
npm start
```

### Linting

```bash
# Verificar calidad del código
npm run lint
```
